Lab 1: Intro to Logic Simulation

-----------
DESCRIPTION

In this lab, the user turns switches on and off to produce 3 main different outputs. The switches represents inputs from the user, that is transmitted through 
gates with different logic. The main outputs are Decimals and Lights.
-----------
FILES

-
Lab1.lgi

This file includes the circuits and diagrams of the lab. 
-----------
INSTRUCTIONS

This file is intended to be run using the Multimedia Logic Application known as (MML). Use the input switches to produce the outputs required for the lab.
