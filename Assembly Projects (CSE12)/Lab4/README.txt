Lab 4: Functions and Graphics

-----------
DESCRIPTION

In this program, you will use MIPS memory-map IO, macros, and stack to draw crosshairs and pixels with different colors inputted by the test file. 
-----------
FILES

-
lab4.asm

This file includes the assembly code for this assignment.
-
lab4_f21_test.asm

This file includes the tests in order to run lab4.asm.
-----------
INSTRUCTIONS

This file is intended to be run using Mars Mips program. This program needs to be directly used with the lab4 test file. Assemble lab4.asm file
and connect bitmap display to MIPS. Assemble lab4_f21_test.asm and program will run.
